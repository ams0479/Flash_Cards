Intro to Programming Flash Cards

Javascript

  • a programming language used to make webpages interactive
  • supported by all modern browsers
  • founded by Brendan Eich
  • official name is ECMAScript (European Computer Manufacturer's Association)

Operators

Arithmetic Operators (returns the result of the action performed, a # or NaN or Infinity)
  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
Assignment Operators (changes the value of the variable on the left of the operator)
  • = Same as: x = y
  • += Same as: x = x + y
  • -= Same as: x = x - y
  • *= Same as: x = x * y
  • /= Same as: x = x / y
  • %= Same as: x = x % y
Comparison Operators (returns a boolean value)
  • === is equal to
  • !== is not equal to
  • > greater than
  • >= greater than or equal to
  • < less than
  • <= less than or equal to

Variables

  • containers for storing data values
  • to initialize a variable use the var keyword; var = myNumber
  • must be identified with unique names
  • names must begin with a letter, or $, or_
  • names are case sensitive
  • reserved words (like JS keywords)can't be used as names

Functions

  • aka a "subprogram"
  • a block of code that performs an action and returns a result
  • can take an argument
  • syntax for declaring functions:
  • function name ([param[, param[, ..]]]) { statements }

Methods

  • an action that a number can perform or have taken on it
  • a property containing a function definition
  • Ex: 48432.78.toExponential(); returns 4.843278e+4

Arguments

  • the real values passed to (and received by) the function
